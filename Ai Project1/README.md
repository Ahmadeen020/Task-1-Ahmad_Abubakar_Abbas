# Rule-Based AI Chatbot

This project is my first Artificial Intelligence training project.  
It is a simple rule-based chatbot built with Python using control flow, input sanitization, and dictionary-based response matching.

## Features
- Responds to predefined greetings and basic inputs.
- Uses `while True` for a continuous loop.
- Cleans user input with `.lower().strip()`.
- Returns a default response for unknown inputs.
- Exits cleanly when the user types `exit`.

## Technologies Used
- Python 3

## How to Run
1. Clone the repository.
2. Open the project folder.
3. Run the chatbot file using Python:
   ```bash
   python chatbot.py
   ```

## Example Usage
```text
You: hello
Bot: Hi there!

You: how are you
Bot: I'm doing well, thanks for asking.

You: exit
Bot: Goodbye!
```
