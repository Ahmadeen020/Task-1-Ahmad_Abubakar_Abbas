# AI Recommendation System

## Project Description

This project is a simple AI Recommendation System developed in Python.

The system collects user interests, compares them with predefined item categories, and recommends the most relevant items based on matching preferences.

## Features

- Accepts user interests as input
- Uses pattern matching logic
- Calculates recommendation scores
- Sorts recommendations by relevance
- Displays personalized recommendations

## Technologies Used

- Python

## Example

Input:

python, cybersecurity

Output:

Cybersecurity Basics (Match Score: 1)
Python for Beginners (Match Score: 1)

## Concepts Demonstrated

- User Input Handling
- Lists and Dictionaries
- Loops
- Conditional Statements
- Recommendation Logic
- Pattern Matching
